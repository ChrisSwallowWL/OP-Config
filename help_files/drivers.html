Connection Methods

There are two ways to connect to the TCB: directly with a Micro-B USB cable, or via Bluetooth using an adapter on the Serial 1 connector. Dipswitch #5 on the TCB determines which port it will attempt to communicate over: if Dipswitch #5 is in the On position, it will communicate over USB; if Off then communication will take place over the Serial 1 port.

USB Drivers - For Direct Connection

The TCB uses a genuine FTDI brand USB chip for direct connection to the computer via USB cable. The drivers you need are the FTDI Virtual COM Port (VCP) drivers. These can be downloaded for free from FTDI.

For links to the drivers and step-by-step instructions with photos, please see the Open Panzer Wiki - USB Drivers page.

Bluetooth Drivers - for Wireless Connection

Windows already includes the necessary Bluetooth drivers so nothing needs to be installed. However Windows will need to pair with the device, see the Open Panzer Wiki - Bluetooth page for more information on this process, as well as links to the recommended Bluetooth adapter compatible with the TCB.
