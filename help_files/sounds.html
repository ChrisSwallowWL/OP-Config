Sound Settings

Sound Device Selection

Presently the TCB is only compatible with the Benedini sound cards - tested to work with the TBS Mini but it should also be compatible with the TBS Micro. Setup is simple, connect Prop 1, 2 & 3 from the TBS Mini to Prop 1, 2, & 3 on the TCB. Then add your sounds to the TBS Mini using the Benedini TBS-Flash program. Sounds must be organized in a specific order, which is however simple to do. The TCB can trigger up to 13 sounds on the Benedini device, in addition to all engine sounds (startup, shutdown, idle, accelerate, etc):

  • Turret rotation
  • Cannon fire
  • Machine gun fire
  • Cannon hit
  • Machine gun hit
  • Vehicle destroyed
  • Headlight (switch) on/off sound
  • Squeak 1
  • Squeak 2
  • Squeak 3
  • Custom User Sound 1
  • Custom User Sound 2
  • Repair sound
For complete instructions on how to set up your TCB sound card, please see the Open Panzer Wiki - Sound page.

Squeak Settings

The TCB allocates 3 sound slots to random squeaks (Squeak 1, 2 and 3). When the vehicle is moving, the TCB will trigger each squeak sound randomly within certain time windows that you can adjust. To understand how the random trigger works, let's take an example. Suppose we set the Minimum time interval for Squeak 2 to 2000 mS (2 seconds). Suppose we set the Maximum time interval to 5000 mS (5 seconds). The TCB will pick a random length of time that is at least 2 seconds but no more than 5 seconds. It will wait for this random length of time, and when that time is up, it will cause the Squeak 2 sound to play. Then it will select a new random length of time that again can be any value at least 2 seconds long but no longer than 5 seconds. In other words, the Minimum interval is the minimum length of time that will ever pass before a squeak repeats itself while the Maximum interval is the maximum length of time before a squeak will repeat itself; but the actual length of time between those two extremes is selected at random.

This random selection process continues for all three squeaks simultaneously so long as the vehicle is moving.

Enabled, Min, Max

Enable or disable each squeak, and set a minimum and maximum interval for it to repeat.

Minimum Speed Before Squeaking

Set the minimum vehicle speed that must be reached before the random squeaking process can begin. It can sound unrealistic if your tank has barely started moving and already it is squeaking.

Other Sound Settings

Enable Headlight Sound

Check this option to automatically play the Headlight (Switch) On/Off sound whenever any trigger turns on or off the headlights (Light 1).

Enable Turret Sound

Check this option to enable the turret rotation sound, uncheck to disable.
